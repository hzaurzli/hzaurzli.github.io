Development logs

MicroWorldOmics

Introduction:

MicroworldOmics (a convenient software for microscopic world analysis and visualization), a desktop software developed based on PYQT (version=5), R (version=4.0)，Python (version=3.6) and JavaScript, which aims to make new exploration and contribution to the development of bioinformatics. Realizing data analysis and visualization in the fields of algorithm statistics, sequence analysis, microbiomics , virology,image recognition etc.


MicroWorldOmics V1.3 2024.5

MicroWorldOmics The UI was redesigned to address the unstretchability of the UI page, and the main page was redesigned,finally,defense system analysis was included in MicroWorldOmics V1.3.


MicroWorldOmics V1.2 2023.12

MicroWorldOmics V1.2 incorporates a number of analysis modules, mainly for viruses and microbiomes: multi-language software covering python, R, C++, etc.


MicroWorldOmics V1.1 2023.1

The first version of the software was called "QTBioRZ",A user-friendly interface visualization software,the main functions are following:

a. Blastn and Blastp are for DNA or protein sequences searching.

b. Counting microorganism colonies.

c. Calculating anti-peptides activity.

d. Identifing phages lysogeny.
